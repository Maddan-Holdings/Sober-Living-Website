# GrapesJS Static Site Builder

A visual drag-and-drop website builder powered by GrapesJS, hosted on GitHub Pages as a pure static site (no Jekyll).

## How It Works

- **`/builder.html`** — The full website builder (GrapesJS editor)
- **`/index.html`** — The live homepage (generated by the builder or manually edited)
- **`.nojekyll`** — Disables Jekyll processing on GitHub Pages

## Builder Features

- Drag and drop blocks: sections, heroes, features, pricing, testimonials, headers, footers, buttons, text, and more
- Full style editor with typography, dimensions, decorations, flex, and transforms
- Multi-page system: create, rename, delete, and switch between pages
- Button linking: external URLs or internal page navigation
- Device preview: Desktop, Tablet, Mobile
- Autosave to localStorage
- Publish directly to GitHub Pages via API or download as ZIP
- Import / Export HTML
- Undo / Redo, Canvas clear, Preview mode

## Publishing

1. Open `builder.html` from your GitHub Pages URL
2. Save your GitHub Personal Access Token in Menu → Settings
3. Click **Publish** in the top right to push changes live

## Local Development

Simply open `builder.html` in a browser or serve via any static file server.
